===============================================================================
                                TROUBLESHOOTING
===============================================================================

If login fails:
1. Check if users exist in database
2. Verify password hashes are correct
3. Check user_roles table for permissions
4. Ensure USR_ADMIN_USERS table has admin entry
5. Check PHP error logs for session issues

Common Issues:
- Session data null: User not logged in
- Array offset error: Missing logged_in session data
- 500 errors: Check PHP logs for specific errors
