🔧 S3 API Methods

GET
/s3/list
List all files in S3 bucket
curl http://localhost:3000/files
GET
/s3/get/{filename}
Get/Download file from S3
curl http://localhost:3000/download/filename.txt
PUT
/s3/put
Upload/Put file to S3
curl -X POST -F "file=@yourfile.txt" http://localhost:3000/upload
DELETE
/s3/delete/{filename}
Delete file from S3
curl -X DELETE http://localhost:3000/delete/filename.txt
